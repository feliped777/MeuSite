Calculadora de Terrenos

Este projeto foi criado para ajudar minha sogra que é corretora de terrenos e imóveis e precisava de um sistema que calculasse para ela, além de precisar transformar metros quadrados para hectares.
A calculadora se encontra no site: Sayonara Sales Imóveis
